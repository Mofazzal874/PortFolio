CSS media query

Media query is very important for mobile devices

Flexbox.css also have some examples on media query , check that..


@media media-type and (media-feature/media query selector) {
        /* CSS rules to apply when the media query conditions are met */
    }

Types of Devices(media-type)

  • Screen-desktop ,phone
  • print device - print the page
  • speech device-Text to speech , screen reader
  • all - to select all types of devices

Example:


        @media all and (max-width:500px){
            body{
                color:blue ; 
            }
        }

    N.B: here you can omit the "all and" and write like:
    @media (max-width:500px){
        body{
            color:blue ; 
        }
    }
    because by default "all and" is putter there you u don't write it explicitly.

Combining Multiple Conditions:


        /* Styles applied when the viewport width is between 600px and 1200px, and the device is in landscape orientation */
@media screen and (min-width: 600px) and (max-width: 1200px) and (orientation: landscape) {
    body {
        font-size: 16px;
    }
}

Using Media Types:


        /* Styles applied when printing the document */
@media print {
    body {
        color: black;
    }
}

Using logical Operators


        /* Styles applied when the viewport width is NOT greater than 768px */
@media not screen and (min-width: 768px) {
    body {
        color: red;
    }
}
